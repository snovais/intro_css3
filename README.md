📘 CSS3 - Resumo Básico

O CSS3 (Cascading Style Sheets) é a linguagem usada para estilizar páginas web, definindo cores, layouts, fontes e animações. Ele trabalha junto com o HTML para melhorar a aparência e a experiência do usuário.

🎨 Principais funcionalidades
Cores e fundos: suporte a RGB, HEX, HSL, gradientes (linear-gradient, radial-gradient)
Tipografia: controle de fontes, tamanhos, espaçamentos e uso de fontes externas (@font-face)
Layout moderno:
Flexbox → alinhamento e distribuição de elementos em uma dimensão
Grid → criação de layouts em duas dimensões
Responsividade:
Uso de media queries para adaptar o layout a diferentes telas (mobile, tablet, desktop)
Animações e transições:
transition para mudanças suaves
@keyframes para animações mais complexas
Sombras e efeitos visuais:
box-shadow, text-shadow, border-radius

🚀 Conclusão

O CSS3 permite criar interfaces modernas, responsivas e interativas, sendo essencial no desenvolvimento front-end.
